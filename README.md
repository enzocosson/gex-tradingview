# GEX Levels TradingView

Indicateur automatique affichant les niveaux GEX pour ES et NQ.

## Installation

1. Cloner le repo
2. Créer `.env` avec votre clé GexBot
3. `pip install -r requirements.txt`
4. `python test_api.py`
5. `python update_gex.py`
6. Configurer GitHub Actions avec secret `GEXBOT_API_KEY`
